## ğŸ“Š **Key Findings Summary**

* Risk score of **8/10 (High)** due to lack of critical controls
* No encryption used for financial or customer data
* All employees had access to sensitive internal databases
* No intrusion detection system (IDS) implemented
* No backups or disaster recovery plan
* Weak password policy + no centralized password manager
* Legacy systems lacked documented maintenance procedures
* PCI DSS and SOC requirements **not met**
* GDPR only **partially compliant**

---

## ğŸ› ï¸ **Recommendations**

### **High Priority**

* Implement encryption for cardholder and customer data
* Enforce least privilege & separation of duties
* Deploy IDS for threat monitoring
* Establish disaster recovery and data backup procedures
* Set up a centralized password management system
* Strengthen password policy to modern standards
* Classify all IT assets and maintain inventory

### **Medium / Low Priority**

* Create schedule for legacy system maintenance
* Improve access control documentation
* Continue monitoring physical security and CCTV systems
